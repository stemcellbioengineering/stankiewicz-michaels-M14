# stankiewicz-michaels-M14
iPSC-T cells at DP maturation day M14, PSC2 media, iPS11 cells

## Overview
This dataset is published as part of [Jones & Salim et al.](https://www.biorxiv.org/content/10.1101/2024.10.29.620998v1)

## Installations/Reproducibility
1. Setup a conda environment with environment.yml
2. Thresholdpy is required to reprocess from the raw data; fetch with git.
```bash
git clone https://github.com/jonesr18/ThresholdPy
```
3. Download .h5md data from https://doi.org/10.17632/k3xtwnzpyn.2 and put into /data/ folder
